# 🧭 Qibla Direction Calculator

Application professionnelle de calcul de direction Qibla avec support UTM, GPS et stakeout topographique.

## 🌟 Fonctionnalités

- 📐 **Calcul Qibla précis** (formule Great Circle Bearing)
- 🌐 **Coordonnées UTM** (11 zones supportées)
- 📡 **Capture GPS automatique**
- 🗺️ **Carte interactive** (Leaflet + Satellite Esri)
- 📏 **Ligne d'implantation** topographique (Stakeout)
- 🧭 **Boussole temps réel** (avec capteurs mobiles)
- 📤 **Export multi-formats** : PDF, DXF (AutoCAD), KML, CSV, TXT
- 🌍 **Multilingue** : Français, English, العربية
- 🎨 **Logo société personnalisable**
- 🔐 **Protection par mot de passe**
- 📱 **PWA installable** sur mobile

## 🚀 Utilisation

👉 **Application en ligne** : https://votre-nom.github.io/qibla-direction-calculator

🔑 Mot de passe : *contact admin*

## 🛠️ Technologies

- HTML5 / CSS3 / JavaScript (Vanilla)
- Leaflet.js (cartographie)
- Proj4.js (projections UTM)
- jsPDF + html2canvas (export PDF)
- Service Worker (mode offline)

## 📱 Installation sur mobile

1. Ouvrir le lien dans Chrome/Safari
2. Menu → **"Ajouter à l'écran d'accueil"**
3. L'app fonctionne comme une vraie application native
